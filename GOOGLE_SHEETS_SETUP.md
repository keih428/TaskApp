# Google Sheetsデータフォーマット

タスク管理アプリと連携するGoogle Sheetsのデータフォーマットです。

## 列の構成

| 列 | 項目名 | データ型 | 説明 | 例 |
|----|--------|----------|------|-----|
| A | ID | テキスト | タスクの一意識別子 | task001 |
| B | タスク名 | テキスト | タスクの名前（必須） | プレゼン資料作成 |
| C | 期限 | 日付 | タスクの期限（YYYY-MM-DD形式） | 2025-01-15 |
| D | 登録日 | 日付 | タスクの登録日（YYYY-MM-DD形式） | 2025-01-01 |
| E | 重要 | ブール値 | 重要フラグ（TRUE/FALSE） | TRUE |
| F | 緊急 | ブール値 | 緊急フラグ（TRUE/FALSE） | TRUE |
| G | 詳細 | テキスト | タスクの詳細情報 | 30ページの資料を作成する |

## サンプルデータ

```
ID          タスク名                期限          登録日        重要    緊急    詳細
task001     プレゼン資料作成        2025-01-15    2025-01-01    TRUE    TRUE    30ページの資料を作成する
task002     メール返信              2025-01-08    2025-01-05    FALSE   TRUE    顧客からの問い合わせに回答
task003     戦略会議準備            2025-01-20    2025-01-03    TRUE    FALSE   Q2の戦略を検討
task004     SNS投稿                 2025-01-10    2025-01-06    FALSE   FALSE   週次の投稿を準備
```

## Google Sheets設定手順

### 1. スプレッドシートの作成
1. Google Sheetsで新しいスプレッドシートを作成
2. 1行目にヘッダー（ID, タスク名, 期限, 登録日, 重要, 緊急, 詳細）を入力
3. 2行目以降にタスクデータを入力

### 2. 共有設定
1. スプレッドシート右上の「共有」ボタンをクリック
2. 「リンクを知っている全員」に設定
3. 権限を「閲覧者」に設定（アプリは読み取り専用）

### 3. スプレッドシートIDの取得
スプレッドシートのURLから取得：
```
https://docs.google.com/spreadsheets/d/【ここがスプレッドシートID】/edit
```

例：
```
https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms/edit
```
この場合、IDは `1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms`

### 4. Google Sheets APIキーの取得

1. [Google Cloud Console](https://console.cloud.google.com/)にアクセス
2. プロジェクトを作成（または既存のプロジェクトを選択）
3. 「APIとサービス」→「ライブラリ」を開く
4. "Google Sheets API"を検索して有効化
5. 「APIとサービス」→「認証情報」を開く
6. 「認証情報を作成」→「APIキー」を選択
7. 作成されたAPIキーをコピー
8. （推奨）APIキーを制限：
   - アプリケーションの制限：HTTPリファラー
   - API制限：Google Sheets APIのみ

### 5. アプリでの設定
1. タスク管理アプリを開く
2. 左パネル下部の「Google Sheets設定」セクションを開く
3. スプレッドシートIDを入力
4. APIキーを入力
5. 「設定を保存」をクリック
6. 「同期」ボタンでデータを読み込み

## 注意事項

- アプリは現在**読み取り専用**です（スプレッドシートからアプリへの同期のみ）
- アプリで追加・編集したタスクはローカルストレージに保存されます
- スプレッドシートのデータを変更した場合、「同期」ボタンで再読み込みしてください
- APIキーは安全に管理してください
- 無料のGoogle Cloud Platformアカウントには使用制限があります

## トラブルシューティング

### 「スプレッドシートの取得に失敗しました」エラー
- スプレッドシートIDが正しいか確認
- APIキーが正しいか確認
- スプレッドシートの共有設定が「リンクを知っている全員」になっているか確認
- Google Sheets APIが有効化されているか確認

### データが読み込まれない
- スプレッドシートの1行目がヘッダー行になっているか確認
- 日付がYYYY-MM-DD形式になっているか確認
- 重要・緊急フラグがTRUE/FALSEになっているか確認
