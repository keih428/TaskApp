# TaskApp
A web application for task management using the Eisenhower Matrix.

## 概要
このアプリケーションは、アイゼンハワーマトリックス（緊急度と重要度の4象限）を使用してタスクを管理するWebアプリケーションです。

## 機能

### レイアウト
- ページは左右に1:3の割合で分割
- 左側：タスクの新規登録・編集フォーム
- 右側：4象限のマトリックス（緊急度×重要度）

### タスク管理
- **タスク要素**：
  - タスク名（必須）
  - 期限（必須）
  - 登録日（自動設定）
  - 重要フラグ
  - 緊急フラグ
  - 詳細欄

### 表示機能
- **4象限分類**：
  - 緊急かつ重要
  - 緊急だが重要でない
  - 重要だが緊急でない
  - 緊急でも重要でもない

- **ソート機能**：
  - 期限が近い順
  - 登録日が古い順
  - 登録日が新しい順

- **ビジュアル機能**：
  - 期限までの日数に応じたカラーグラデーション
    - 期限切れ：赤
    - 今日：オレンジ
    - 1-3日：黄色
    - 4-7日：緑
    - それ以上：グレー

### タスク操作
- **チェックボックス機能**：
  - チェックを入れると3秒後に自動的にアーカイブ
  - 3秒以内にチェックを外せばアーカイブをキャンセル
  - アーカイブされたタスクはArchiveシートに移動され、完了日時が記録されます
  
- **編集機能**：
  - カード（チェックボックス以外）をクリックで編集画面に移動

### データ管理
- Google Sheetsをデータベースとして使用
- **Activeシート**: 未完了のタスクを保存
- **Archiveシート**: 完了したタスクを保存（完了日時付き）
- マルチデバイス対応（どのデバイスからも同じデータにアクセス可能）
- リアルタイム同期（タスクの追加・編集・アーカイブが即座に反映）

## 使い方

### 基本的な使い方
1. `index.html` をブラウザで開く
2. 左パネルでタスクを入力
   - タスク名と期限は必須
   - 重要・緊急のチェックボックスを選択
   - 詳細を入力（任意）
3. 「保存」ボタンをクリック
4. 右パネルの4象限にタスクが表示される

### タスクの編集
1. タスクカード（チェックボックス以外）をクリック
2. 左パネルに情報が読み込まれる
3. 修正後「更新」ボタンをクリック

### タスクの削除
1. タスクカードのチェックボックスにチェックを入れる
2. 3秒間待つと自動的にアーカイブされる
3. 3秒以内にチェックを外すとキャンセル可能
4. アーカイブされたタスクはArchiveシートに完了日時と共に保存される

### Google Sheets連携（必須）

このアプリはGoogle Sheetsをバックエンドデータベースとして使用します。複数のデバイスから同じタスクデータにアクセスできます。

#### 前提条件
1. Googleアカウント
2. Google Sheets
3. Google Apps Script（無料）

#### セットアップ手順

詳細は [GOOGLE_SHEETS_SETUP.md](GOOGLE_SHEETS_SETUP.md) を参照してください。

**簡易手順**:
1. Google Sheetsで新しいスプレッドシートを作成
2. 「拡張機能」→「Apps Script」を選択
3. `google-apps-script.js` の内容をコピー＆ペースト
4. 「デプロイ」→「新しいデプロイ」→「ウェブアプリ」として設定
   - 次のユーザーとして実行: 自分
   - アクセスできるユーザー: 全員
5. デプロイされたWeb App URLをコピー
6. タスク管理アプリの「Google Sheets設定」にURLを貼り付け
7. 「設定を保存」をクリック

これで、どのデバイスからアクセスしても同じタスクが表示されます。
