# TaskApp
A web application for task management using the Eisenhower Matrix.

## 概要
このアプリケーションは、アイゼンハワーマトリックス（緊急度と重要度の4象限）を使用してタスクを管理するWebアプリケーションです。

## 機能

### レイアウト
- ページは左右に1:3の割合で分割
- 左側：タスクの新規登録・編集フォーム
- 右側：4象限のマトリックス（緊急度×重要度）

### タスク管理
- **タスク要素**：
  - タスク名（必須）
  - 期限（必須）
  - 登録日（自動設定）
  - 重要フラグ
  - 緊急フラグ
  - 詳細欄

### 表示機能
- **4象限分類**：
  - 緊急かつ重要
  - 緊急だが重要でない
  - 重要だが緊急でない
  - 緊急でも重要でもない

- **ソート機能**：
  - 期限が近い順
  - 登録日が古い順
  - 登録日が新しい順

- **ビジュアル機能**：
  - 期限までの日数に応じたカラーグラデーション
    - 期限切れ：赤
    - 今日：オレンジ
    - 1-3日：黄色
    - 4-7日：緑
    - それ以上：グレー

### タスク操作
- **チェックボックス機能**：
  - チェックを入れると3秒後に自動削除
  - 3秒以内にチェックを外せば削除キャンセル
  
- **編集機能**：
  - カード（チェックボックス以外）をクリックで編集画面に移動

### データ管理
- ローカルストレージに自動保存
- Google Sheets連携（オプション）

## 使い方

### 基本的な使い方
1. `index.html` をブラウザで開く
2. 左パネルでタスクを入力
   - タスク名と期限は必須
   - 重要・緊急のチェックボックスを選択
   - 詳細を入力（任意）
3. 「保存」ボタンをクリック
4. 右パネルの4象限にタスクが表示される

### タスクの編集
1. タスクカード（チェックボックス以外）をクリック
2. 左パネルに情報が読み込まれる
3. 修正後「更新」ボタンをクリック

### タスクの削除
1. タスクカードのチェックボックスにチェックを入れる
2. 3秒間待つと自動的に削除される
3. 3秒以内にチェックを外すとキャンセル可能

### Google Sheets連携（オプション）

#### 前提条件
1. Google Cloud Platformでプロジェクトを作成
2. Google Sheets APIを有効化
3. APIキーを作成
4. スプレッドシートを作成し、共有設定を「リンクを知っている全員」に設定

#### スプレッドシートの形式
以下の列でデータを作成してください：

| A列 | B列 | C列 | D列 | E列 | F列 | G列 |
|-----|-----|-----|-----|-----|-----|-----|
| ID | タスク名 | 期限 | 登録日 | 重要 | 緊急 | 詳細 |
| task1 | 会議準備 | 2025-01-05 | 2025-01-01 | TRUE | TRUE | 資料作成 |

#### 設定手順
1. 左パネル下部の「Google Sheets設定」セクションを開く
2. スプレッドシートIDを入力（URLの`/d/`と`/edit`の間の文字列）
3. Google Sheets APIキーを入力
4. 「設定を保存」をクリック
5. 「同期」ボタンでデータを読み込み
